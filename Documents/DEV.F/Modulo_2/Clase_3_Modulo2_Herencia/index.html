La herencia en CSS es un concepto fundamental que permite a los elementos hijos heredar estilos.

  1. La herencia se refier a como los elementos hijos pueden recibir estilos definidos en sus elementos padre.
  2. Existen propiedades que se heredan por defecto y propiedades que no se heredan por defecto
  3. La herencia se puede forzar con "inherit"
  4. Y también se puede prevenir la herencia con "initial"
